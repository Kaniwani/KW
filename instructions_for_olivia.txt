TO OPEN KW:
---------------------------
Go to virtual environment, and run source bin/activate
Go to KW, make manage.py executable and run ./manage.py runserver

TO GET SRS WORKING:
---------------------------
Open an instance of KW

Go to virtual environment in two terminals (you can do this!)
Open root directory of project in both.

In one, execute:
	celery -A KW worker --loglevel=info

In the other, execute:
	celery -A KW beat --loglevel=INFO
